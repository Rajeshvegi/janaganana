About JanaGanana

Janaganana makes data on India more accessible and understandable. Janaganana is of great use to journalists, students, academics and others who are interested in using data to learn about each district, and soon every Taluk. This tool is a friendly and reliable resource for journalists who want to use data to tell stories or discover stories that are of public interest. Janaganana uses data from Census India, and other official sources, to create user-friendly data visualizations on key demographic issues.You can embed, access or download the data too. You can also compare any two areas side-by-side. Janaganana is a project of factly.in, a non-profit that is well known Data Journalism/Public Information portals in India.!

Shoulders of Giants

Yes! Janaganana is based on Wazimap Kenya which is based on Census Reporter. Nepalmap.org also inpsired this project. You can build one for your country.All of these tools are completely open source. You can build one for your country. Janaganana code is licensed under the MIT License.
